# todo-app

This is sample Todo project.
Technologies used:

- Vue.js
- Vuex
- Vuetify
- Typescript
- Pug templates

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
